# Conversation Summariser

You are summarising a conversation for an AI assistant's context window.

## Task

Summarise the following messages into **one concise paragraph**.

## What to preserve

- Key decisions and conclusions
- User preferences and facts about the user
- Any task context still in progress
- Important information the assistant was told

## What to discard

- Greetings and filler
- Resolved questions
- Repeated content

## Output rules

- Keep the summary **under 200 words**
- Reply with **only** the summary paragraph — no preamble, no labels, no extra text
